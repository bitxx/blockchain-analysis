# 创建项目  
## 概述  
官方和社区已有项目应用地址：http://truffleframework.com/boxes/metacoin  
## 安装  
* 从头开始项目建立空项目  
    1. 创建和初始化：  
        ```bash
        mkdir testproject
        cd testproject
        truffle init
        ```  
        ![image](/doc/img/truffle/3-1.png)  
    2. 目录结构：  
        ```bash
        find . -print | sed -e 's;[^/]*/;|____;g;s;____|; |;g’ //mac环境
        ```
